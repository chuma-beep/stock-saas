# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-12

### Added
- Initial release
- Stock comparison functionality
- AI-powered analysis using Groq
- Interactive charts (Area & Line)
- Date range presets for seasonal analysis
- Live price ticker on homepage
- Export to CSV functionality
- PostgreSQL caching for stock data
- Integration with Alpha Vantage API

### Tech Stack
- Backend: Go + Gin + PostgreSQL
- Frontend: Next.js 15 + TypeScript + Tailwind
- AI: Groq LLaMA 3.3 70B
- Hosting: Render (backend) + Vercel (frontend)

## [Future]

### Planned Features
- User authentication
- Save favorite comparisons
- Compare 3-4 stocks simultaneously
- Email alerts for price changes
- More AI insights and chat interface
- Advanced charting options
